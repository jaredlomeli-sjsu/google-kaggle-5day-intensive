## Tech Stack

- **Google ADK** — multi-agent workflow orchestration
- **Gemini API** — language model backbone (`gemini-flash-latest`)
- **Flask** — lightweight Python web server
- **Claude Code** — AI coding assistant with custom skills
- **Google Cloud Run** — serverless deployment

## Getting Started

Each day folder contains its own setup instructions. Projects that require API keys use a `.env` file — copy the provided `.env.example` and fill in your own `GOOGLE_API_KEY` from [Google AI Studio](https://aistudio.google.com/).
