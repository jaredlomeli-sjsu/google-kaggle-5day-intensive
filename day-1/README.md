# Day 1 — Introduction & Cloud Run Demo

## Atmospheric Effects Showcase

The opening day featured a visual effects web app deployed on **Google Cloud Run**, demonstrating the end-to-end Google Cloud deployment workflow.

**Live URL:** https://atmospheric-effects-showcase-128695111988.us-west2.run.app/

This app was used as a first look at serverless deployment on Google Cloud — write code, containerize, and ship to a public URL with a single `gcloud` command.
